前言
spring-boot-vue项目是基于spring boot + vue.js开发，用于学习spring boot和vue.js。
项目介绍
spring-boot-vue项目，目前实现登录，搜索历史。
插件依赖
JAP: Java Persistence API,Java持久层API，是JDK 5.0注解或XML描述对象－关系表的映射关系，并将运行期的实体对象持久化到数据库中。
H2: 关系数据库，实现数据持久化
Lombok: 简化消除冗余代码
CORS:跨域资源共享
